The database consists of 4 tables - Students, Courses, Faculty and Enrollment.

This website allows you to insert entries in the database as well as query some information regarding the database:

  • Insert An Entry
    •
    1. Insert an entry in Students Table
    2. Insert an entry in Courses Table
    3. Insert an entry in Faculty Table
    4. Insert an entry in Enrollment Table
  • Query Some Information
    •
    1. Ask the number of students in a course
    2. Ask the average in a particular exam of a course
